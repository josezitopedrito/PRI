Abstract

Este trabalho semanal consistia em desenvolver um servidor que disponibiliza as informações acerca de um conjunto de obras e respetivos compositores armazenados numa base de dados não relacional em MongoDB, com o auxílio de NodeJS e de ExpressJS.

O servidor é capaz de receber pedidos de GET para o conjunto total de obras, para a informação individual de cada obra, para a informação acerca do período, ano de criação e compositor da obra, para o conjunto total de compositores e para as obras de cada compositor.

Deliverables

  • Diretoria que contém o TPC7
  • Ficheiro XML que contém a informação deste PR
  • Stylesheet que contém a informação deste PR
  • Ficheiro em JSON que contem a informação que depois é apropriada pela base de dados em mongo.
