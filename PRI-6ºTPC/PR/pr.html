Abstract

Este trabalho semanal consistia em desenvolver um servidor que disponibiliza as informações acerca de um conjunto de filmes armazenados numa base de dados não relacional em MongoDB, com o auxílio de NodeJS e de ExpressJS.

O servidor é capaz de receber pedidos de GET para o conjunto total de filmes e para a informação individual de cada filme. O servidor também é capaz de receber pedidos de POST para acrescentar filmes. Por fim o servidor é capaz de receber pedidos de DELETE para apagar filmes.
