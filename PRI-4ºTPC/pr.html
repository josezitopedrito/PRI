Abstract

Na mesma diretoria deste manifesto estão mais 6 ficheiros e 1 diretoria.

Nos ficheiros pr.xml e pr.xsl encontra-se a informação necessária para elaborar esta página em HTML.

No ficheiro w3.css encontra-se codigo css para ser implementado nas páginas de individuais de cada música.

O ficheiro datasetGen.xsl contém a stylesheet que criou as páginas em XML para cada música.

O ficheiro doc2html.xsl contém a stylesheet necessária para passar cada ficheiro individual xml para um ficheiro individual html.

O ficheiro arq-son-EVO.xml é o ficheiro em XML que conté informação de todas as músicas do arquivo sonoro.

A diretoria data contém todos os ficheiros em XML individuais de todas as músicas no arquivo sonoro.

O ficheiro arq.js contém o script necessário para criar o servidor.

Este trabalho semanal consistia em desenvolver um servidor com nodeJS que disponibiliza as informações acerca das músicas com o auxílio do nodeJS.

Deliverables

  • Ficheiro XML do manifesto.
  • Stylesheet relativa à elaboração do HTML do manifesto.
  • Stylesheet que elabora os ficheiros individuais em XML a partir do arquivo sonoro.
  • Stylesheet que elabora o HTML a partir dos ficheiros individuais XML.
  • Ficheiro XML que apresenta as caraterísticas do arquivo sonoro.
  • Diretoria com os ficheiros individuais XML.
